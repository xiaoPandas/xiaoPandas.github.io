VMware虚拟机安装相关

发表于 2018-02-22 | 分类于 错误汇总 |

虚拟机安装linux的时候，出现“无法检测此光盘映像中的操作系统，您需要指定要安装的操作系统”的问题。

解决：

  1. 不选择“安装程序光盘映像文件”，要选择“稍后安装操作系统”；
  2. 下一步直至“已准备好创建虚拟机”(中间选项自行设置)；
  3. 点击“自定义硬件”–》“新CD/DVD”；
  4. 在连接的地方选择“使用ISO映像文件”，然后选择自己准备好的映像就行了。

忘记root密码

解决：

  1. 重启ubuntu，随即长按shift进入grub菜单；
  2. 选择recovery mode，回车确认；
  3. 在Recovery Menu中，选择“Root Drop to root shell prompt”，回车确认；
  4. 进入shell界面，使用passwd命令重新设定密码：passwd root，回车确认；
  5. 输入新密码；
  6. 再次确认新密码；
  7. 看到密码设定成功提示：passwd: password updated successfully；
  8. 重启系统进入GUI界面：sudo reboot；

备注：
在修改密码的时候，或许会出现 authentication token manipulation error 导致修改不成功，可能是在liveCD中修改文件导致系统重启之后发现数据被破坏，所以给Read-Only了。于是使用mount命令重新挂载成读写模式，命令如下：#mount -o remount rw /，然后再修改密码就ok。
