ubuntu安装

发表于 2018-02-22 | 分类于 Linux |

“无法检测此光盘映像中的操作系统”

安装时，出现“无法检测此光盘映像中的操作系统，您需要指定要安装的操作系统”的错误。

解决：

  1. 不选择“安装程序光盘映像文件”，要选择“稍后安装操作系统”；

  2. 下一步直至“已准备好创建虚拟机”(中间选项自行设置)；

  3. 点击“自定义硬件”–>“新CD/DVD”；

  4. 在连接的地方选择“使用ISO映像文件”，然后选择自己准备好的映像就行了。

忘记密码

解决：

  1. 重启 ubuntu，随即长按 shift 进入 grub 菜单；

  2. 选择第二项 Advanced options for Ubuntu；

  3. 选择recovery mode，按 e 进入命令行；

  4. 在Recovery Menu中，选择“Root Drop to root shell prompt”，回车确认；

  5. 找到有 recovery nomodeset 的行，删除 recovery nomodeset，改为 quiet splash rw init=/bin/bash，然后按 F10；

  6. 在命令行输入 passwd + 用户名，修改密码，若修改成功，则会返回 password updated successfully；

  7. 修改完成，重启系统。
