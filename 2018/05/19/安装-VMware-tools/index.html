安装 VMware tools

发表于 2018-05-19 | 分类于 Linux |

安装步骤

  1. 启动 ubuntu 系统，选择：菜单栏 - 虚拟机 - 安装 VMware Tools，若已安装，则会显示重装 VMware Tools；

  2. 在文件中选择：设备 - VMware Tools，发现其中有一个 VMwareTools-···.tar.gz 的文件，右击，选择提取到，然后提取到某个目录下，我是在『文档』中新建了一个『vmwaretools』文件夹；

  3. 提取之后发现『vmwaretools』中出现『vmware-tools-distrib』文件夹；

  4. 打开终端，切换到 root 用户；『sudo su』

  5. 进入到『vmwaretools』文件夹，输入命令：./vmware-install.pl

  6. 根据提示输入 yes/no，或者直接回车，直到出现 Enjoy，则表示安装成功；

  7. 重启虚拟机。

提醒

如果想让虚拟机和 windows 实现文件拖拽复制，只能在两个环境的桌面上实现拖拽，而不可以从其他目录下操作；其他目录下仅能完成手动复制粘贴。

VMware Tools 文件为空

点击了：虚拟机->安装 VMware tools，CD/DVD 中会加载 VMware tools，但是里面并没有文件。

解决：

  1. 在虚拟机安装目录下找到 linux.iso 文件；『E:\Downloads\Linux\VMware』

  2. 将 linux.iso 拷贝到装 ubuntu 系统的文件目录下；『E:\Downloads\Linux\JingXiang』

  3. 设置硬件选项 CD/DVD(SATA)，将 『使用 ISO 映像文件』改为刚刚拷贝的 linux.iso 文件；

  4. 重启系统，重新点击安装 VMware Tools，就会出现相关文件了。
