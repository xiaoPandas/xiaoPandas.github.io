Typora 使用指南

发表于 2018-07-26 |

使用了有一年的时间的 Cmd Markdown 了，有些地方真的是鸡肋，体验感不太好，本地图片插入要开通会员，不能直接拖入，有几次本地的预览页面和上传服务器后页面显示还不一样，写得都有点烦了，但是一直不知道其他哪个比较好，也就将就着用了。刚好昨天昌哥推荐了 Typora，果断下载安装，看到这个简洁的界面我就喜欢 :happy:

二者基本的用法大都相同，Typora 的功能更加强大，下面记录一下具体用法：

目录

格式：[toc]

效果：

[TOC]

标题

效果：

引用

  • 普通引用，格式：> 危楼高百尺，手可摘星辰

    效果：

    危楼高百尺，手可摘星辰

  • 代码引用，格式：用 `` 括起来「单行」或按三个点

列表

  • 无序列表

    格式：- 无序

  • 有序列表

    格式：1. 有序

  • 任务列表

    格式：- [] 或 - [ x ]

表格

格式：| | | 回车或快捷键 ctrl+T

姓名 年龄
xiaoPandas 3

强调

  • 斜体

    格式 ：*倾斜*或快捷键 ctrl+I

    效果：倾斜

  • 粗体

    格式：**粗体** 或快捷键 ctrl+B

    效果：粗体

  • 下划线

    格式：<u>下划线</u> 或快捷键 ctrl+U

    效果：下划线

  • 删除线

    格式：~~删除线~~

    效果：删除线

链接与图片

  • 本地图片

    格式：直接拖放到编辑页面「如果要上传到服务器本地图片不能显示」

    效果：

  • 在线图片

格式：![name](url)

效果：

  • 链接

    格式：[name](url) 或 <url>
